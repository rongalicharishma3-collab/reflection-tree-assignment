# Daily Reflection Tree Assignment

## Overview
This repository contains the implementation of a structured **Reflection Tree** designed to guide individuals through a process of self-reflection across three psychological axes:

1. Self-Awareness
2. Contribution
3. Growth Mindset

The reflection tree allows users to navigate through questions, decisions, and reflection points to analyze their behavior and mindset.

## Tree Design

The tree consists of several types of nodes:

- **Question Nodes** – collect user responses
- **Decision Nodes** – interpret user responses
- **Reflection Nodes** – provide insight and reframing
- **Bridge Nodes** – transition between reflection axes
- **Summary Node** – provides final reflection

The tree covers three axes in sequence:

1️ Self-awareness  
2️ Contribution  
3️ Growth mindset  

---

## Reflection Axes

### Axis 1 – Self Awareness
Focuses on how individuals react to challenges and whether they reflect on their actions.

### Axis 2 – Contribution
Explores teamwork, collaboration, and how individuals contribute to collective success.

### Axis 3 – Growth Mindset
Examines responses to failure and long-term development motivations.

## Running the Reflection Agent

1. Install Python
2. Run the program:

python reflection_agent.py

The program will load the reflection tree and guide the user through the reflection questions.
